# Into the Maelstrom

A browser-based interactive system that explores narrative through dynamic interface behavior.

## Live Demo

https://sonnyyy77.github.io/Into-the-Maelstrom/

> Best experienced in a standard browser window (not full screen), as the interaction relies on window behavior and layout changes.

## Overview

Into the Maelstrom is an experimental web project where the interface itself becomes part of the experience. Instead of presenting a fixed layout, the system responds to user actions through shifting windows, motion, and timing, creating a sense of instability and loss of control.

## Features

- Event-driven interaction logic using JavaScript  
- Dynamic window behavior and layout changes  
- Real-time response to user input  
- Integration of motion, timing, and sound  
- Narrative expressed through system behavior rather than static content  

## Interaction Model

The experience unfolds as users interact with the browser. Windows move, resize, and respond unpredictably, forming a system where behavior replaces traditional storytelling structures.

## Technical Approach

- JavaScript for interaction logic and event handling  
- Window and DOM manipulation to control layout and behavior  
- Timing and sequencing to guide pacing  
- Audio integration to reinforce interaction feedback  

## Tech Stack

- JavaScript  
- HTML / CSS  
- Web APIs (window control, events, timing)  

## Notes

This project focuses on treating the interface as a behavioral system. Rather than guiding users through content, it builds an experience through interaction patterns, system responses, and temporal changes.
